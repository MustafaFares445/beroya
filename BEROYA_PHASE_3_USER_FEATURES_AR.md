# ملخص ميزات النظام 

هذا الملخص يشرح ما الذي سيلاحظه المستخدم داخل النظام بعد تنفيذ المراحل الثلاث،.

## المرحلة الأولى

- تسجيل الدخول والخروج بطريقة أسرع وأكثر وضوحًا.
- إدارة المستخدمين وصلاحياتهم من شاشة واحدة.
- تنظيم المعارض وأرقام الهواتف المرتبطة بها.
- إدارة السيارات، وإضافة صورها، وتحديث بياناتها بسهولة.
- متابعة طلبات الزبائن بشكل أوضح من داخل النظام.
- إدارة الحسابات الأسبوعية، مع المكافآت والحسومات وحالة الاستلام.

## المرحلة الثانية

- تنظيم العمل العقاري بشكل مرتب حسب المحافظة والمكتب.
- عرض العقارات بحسب النوع والحالة والتفاصيل المهمة.
- إضافة عقار جديد مع الصور والمعلومات الأساسية.
- إرسال طلبات العقار للمراجعة قبل اعتمادها.
- قبول الطلبات وتحويلها إلى عقارات منشورة، أو رفضها مع توضيح السبب.
- تسهيل الوصول إلى بيانات المكاتب وأرقامها والتصنيفات العقارية.

## المرحلة الثالثة

- متابعة حركة المبيعات بشكل أوضح.
- عرض سجل العمليات لمعرفة من قام بالتعديل ومتى.
- استقبال التنبيهات المهمة داخل النظام في مكان واحد.
- إدارة البيع على مراحل: طلب، موافقة، إكمال، أو تحويل إلى تقسيط.
- حفظ تفاصيل العميل والبائع والمشتري في نفس سجل البيع.
- تحديث حالة السيارة والحسابات المرتبطة بها بعد إكمال البيع.
